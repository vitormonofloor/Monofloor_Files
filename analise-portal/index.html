👉 Observação técnica

O campo projetoCidade aceita texto livre, gerando duplicidades como "SÃO PAULO", "São Paulo/São Paulo", "SÃO PAULO / SP", "CEP 05409". Para análises confiáveis, vale normalizar o cadastro usando uma lista controlada de municípios. É um quick-win de qualidade de dados.
